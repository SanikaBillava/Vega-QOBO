# React Web App

## Installation

```bash
npm install
```

## Run Locally

```bash
npm run dev
```

Open http://localhost:5173

## Deploy to Vercel

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel
```

Or connect your GitHub repo to Vercel for automatic deployments.

## Tech Stack
- React
- Vite
- Vercel (deployment)
